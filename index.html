College Project

Malware Detection Based on API Calls Using Machine Learning

A behavioral malware analysis system that monitors program runtime activity via Process Monitor, extracts API call sequences, and applies TF-IDF vectorization to convert behavioral patterns into numerical feature vectors. These vectors are classified using LightGBM to determine whether a program exhibits malicious behavior. Unlike traditional signature-based antivirus solutions, this approach detects unknown and zero-day threats by analyzing what a program does at runtime rather than matching it against known signatures.

Python Scikit-learn LightGBM TF-IDF Tkinter

Object Recogniser

A real-time object recognition system built on a pre-trained YOLO model. The model was configured to interface with the device camera, process live video frames, identify objects within the frame, and display annotated output in real time. Suitable for CCTV and security surveillance applications where automated visual identification is required.

Ongoing · College Project

Threat Intelligence in IoT Networks

An AI-powered threat detection platform designed for IoT network environments. The system uses a Raspberry Pi as an edge controller with ESP32 sensor endpoints, collecting and forwarding network traffic data for anomaly detection using an Isolation Forest algorithm. A custom SIEM-like monitoring dashboard provides real-time alert management, traffic analysis, device health monitoring, and configurable threat thresholds. Currently in active development.

Python FastAPI React Raspberry Pi ESP32 Isolation Forest
